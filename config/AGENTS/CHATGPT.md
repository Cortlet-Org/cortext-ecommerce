# ChatGPT Agent Guidelines

ChatGPT operating within the Cortext repository must respect project philosophy.

## Core Principles

- Vendor-agnostic architecture only.
- No forced integrations.
- No hidden coupling.
- Minimal and explicit design.

## Code Behavior

- Do not introduce unnecessary dependencies.
- Do not assume payment providers.
- Do not add external SDKs without justification.
- Preserve clean separation of concerns.

## Documentation Rules

- Keep language professional and concise.
- Avoid hype or comparative claims.
- Maintain architectural clarity.

## Security

- Never expose secrets.
- Never suggest committing API keys.
- Encourage environment-based configuration.
